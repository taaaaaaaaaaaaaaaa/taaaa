The head element

(1)head要素は、ブラウサーに対する指定と、文書に関するメタ情報など、負荷的な情報を指定するための場所。

(2)<title> sample html </title>

(3)メタデータプロファイルのURLを指定することができた。

The title element

(1)title要素は文章にタイトルをつける要素でそのページのタイトルを記述することのできる要素です。

(2)<title>sampletitle</title>

(3)文章にタイトルをつけることを記述することができました。

The base element

(1)文章内における全ての相対URL基点となるURLを指定する

(2)<base href = "https://example.com">

(3)リンク先がhttps://example.comになる
